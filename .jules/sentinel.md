## 2024-05-24 - [Missing IP Spoofing Protection] | Vulnerability: The Fastify `trustProxy` configuration was missing, making the application susceptible to IP spoofing attacks when deployed behind reverse proxies, as headers like `X-Forwarded-For` were ignored and rate limiting logic could be bypassed. | Learning: Always configure `trustProxy` appropriately to respect upstream reverse proxy metadata securely while ensuring malicious actors cannot override IP addresses directly. | Prevention: Introduce the `APP_TRUST_PROXY` environment variable and ensure the server instance binds its value securely.

The following versions of this project are currently supported with security updates.


| Version       | Supported |
|--------------|-----------|
| Latest minor | :white_check_mark:        |
| Previous minor | :white_check_mark:      |
| Older versions | :x:       |

Only the latest and the immediately previous minor versions receive security fixes.
Older versions are not supported and will not receive patches.

---

## Reporting a Vulnerability

Please **do not report security vulnerabilities through public GitHub issues**.

Instead, use one of the following methods:

- **GitHub Private Vulnerability Reporting**

### What to expect
- **Initial response:** within 72 hours
- **Status update:** within 7 days
- **Fix & disclosure:** coordinated release after validation and patching

If the vulnerability is accepted, we will work on a fix and publish a security advisory.
If it is declined, we will provide a brief explanation.

Thank you for helping keep this project secure.

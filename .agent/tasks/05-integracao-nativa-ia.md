# Task 05: Demonstração Nativa de IA (AI-Driven Development)

## Contexto
O repositório já possui a dependência `@google/generative-ai` no `package.json`, o que é um enorme diferencial. Contudo, isso precisa ser destacado comercialmente como uma feature de valor.

## Objetivos
- Colocar o repositório na nova onda de aplicações "AI-First".
- Mostrar aos desenvolvedores como integrar IA facilmente utilizando Fastify e Bun.

## Itens de Ação
- [ ] Criar uma rota ou CLI de exemplo (ex: `POST /ai/completion` ou um bot simples) usando a biblioteca `@google/generative-ai`.
- [ ] Destacar essa feature no README (Ex: "RAG e IA Generativa integrados por padrão").
- [ ] Documentar o fluxo de configuração da API Key (Gemini) no `.env.exemple`.

## Resultados Esperados
- Novo fator de "hype" (buzzword comercial) ancorado em código real.
- Atração do público de IA que busca formas fáceis de integrar LLMs em backends TypeScript.
